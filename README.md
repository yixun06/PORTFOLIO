# Khew Yi Xun's Portfolio

A personal portfolio website built with React and Vite, with AI-assisted development support and hands-on refinement on layout, interaction, and content.

## About This Project

This portfolio was developed with help from AI tools during implementation and debugging, but the final structure, content, and polish were intentionally refined by hand. The goal was to keep the site fast, modern, and professional while still showing my own technical stack and project experience clearly.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide React

## Features

- Responsive hero section with profile card
- Project showcase with links to live demos and repositories
- Skills and experience sections
- Direct email contact shortcuts
- GitHub Pages deployment ready

## Local Development

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`
3. Build for production:
   `npm run build`

## Deployment

This repository is configured for GitHub Pages deployment through GitHub Actions. After pushing to the `master` branch, the workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml) builds the app and publishes the site.

If GitHub Pages is not enabled yet, set the repository Pages source to GitHub Actions in the repository settings once, then push again.
